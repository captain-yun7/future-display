# 차세대 디스플레이 학습 웹사이트 - 프로젝트 요약

## 📌 프로젝트 개요

**프로젝트명**: 차세대 디스플레이 학습 웹사이트
**발주처**: 충북보건과학대학교 COSS혁신융합대학사업단
**목적**: 디스플레이 기술 교육 및 학습을 위한 인터랙티브 웹 플랫폼

---

## 🛠 기술 스택

### 프론트엔드
- **Framework**: Next.js 15 (React 19)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **애니메이션**: Framer Motion
- **폰트**: Pretendard (한글 최적화)

### 인프라
- **배포**: Vercel
- **데이터베이스**: Neon (PostgreSQL)

---

## 🎯 주요 기능

### 1. 디스플레이 기술 카탈로그
- **CRT** (음극선관) - 브라운관 기술
- **LCD** (액정 디스플레이)
- **LED** (발광 다이오드 디스플레이)
- **OLED** (유기 발광 다이오드)
- **QLED** (퀀텀닷 디스플레이)
- **MicroLED** (초소형 LED 디스플레이)

### 2. 각 기술별 상세 정보
- 작동 원리 (Principle)
- 제조 공정 (Manufacturing)
- 장점 (Advantages)
- 단점 (Disadvantages)
- 활용 분야 (Applications)
- 재미있는 사실 (Fun Facts)

### 3. 인터랙티브 UI/UX
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 부드러운 애니메이션 효과
- 직관적인 네비게이션
- 다크 테마 기반 디자인

## 📁 프로젝트 구조

```
src/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # 루트 레이아웃
│   ├── page.tsx           # 메인 페이지
│   └── displays/[id]/     # 동적 라우팅 (각 디스플레이 상세 페이지)
├── components/
│   ├── layout/            # Header, Footer, Sidebar
│   ├── displays/          # DisplayCard, TechSection
│   ├── ui/                # Button, Card, Input 재사용 컴포넌트
│   └── common/            # ErrorBoundary 등
└── lib/
    ├── displayData.ts     # 디스플레이 기술 데이터
    ├── constants.ts       # 전역 상수
    └── utils.ts           # 유틸리티 함수
```

## 💡 핵심 구현 포인트

### 1. 타입 안전성
```typescript
export interface DisplayTech {
  id: string;
  name: string;
  fullName: string;
  description: string;
  principle: string[];
  manufacturing: string[];
  advantages: string[];
  disadvantages: string[];
  applications: string[];
  images?: string[];
}
```

### 2. 동적 라우팅
- Next.js App Router의 `[id]` 동적 세그먼트 활용
- 각 디스플레이 기술별 독립적인 페이지 생성

### 3. 성능 최적화
- 이미지 최적화 (Next.js Image 컴포넌트)
- 코드 스플리팅 (동적 import)
- Turbopack 빌드 시스템

### 4. 접근성
- 시맨틱 HTML
- ARIA 레이블
- 키보드 네비게이션 지원
- 스크린 리더 지원
